# 国外技术周刊 #142：Agent 原生产品管理——对话即工作

**欢迎打开本期国外技术周刊**：Marcus Moretti 分享了如何用 Agent 原生方式做产品管理——所有工作都在与 Claude Code 的对话中完成；David Epstein 在新书发布之际探讨深度专注的科学；Jasmine Sun 则深入报道了硅谷对 AI 取代白领工作的焦虑。

## 一、本周最受关注的文章

### 1.1 Agent 原生产品管理指南

![A Guide to Agent-native Product Management](https://readwise-assets.s3.amazonaws.com/media/wisereads/articles/a-guide-to-agent-native-produc/cover_image_cUTDXXm.png)

Marcus Moretti 独自运营着 Every 的写作产品 Spiral，他分享了自己的 Agent 原生产品管理方法以及可以注入任何 Agent 的技能。「我不再纠结 SQL 查询里的分号，甚至不再写 ticket。我所有的产品管理工作都在对话中完成——对我来说，是和 Claude Code 对话。对话即工作。」

Marcus Moretti · 11 分钟 | **every.to**

### 1.2 成功的秘诀是「单任务处理」

![The Secret to Success Is 'Monotasking'](https://readwise-assets.s3.amazonaws.com/media/wisereads/articles/the-secret-to-success-is-monot/cover_image.jpg)

为纪念新书《Inside the Box》的发布，《Range》作者 David Epstein 探讨了深度专注的科学。「如果你每天都被通知打断，那么即使这些外部触发器神奇地消失了，你也会无意识地开始自我打断，以维持你习惯的分心节奏。」

David Epstein · 10 分钟 | **The Atlantic**

### 1.3 硅谷正在为永久下层阶级做准备

![Silicon Valley Is Bracing for a Permanent Underclass](https://readwise-assets.s3.amazonaws.com/media/wisereads/articles/the-a-i-fear-keeping-silicon-v/cover_image.jpg)

Jasmine Sun 在最新文章中报道了硅谷内部关于 AI 与工作未来的焦虑对话。「当 20 多岁的旧金山软件工程师讨论逃离永久下层阶级时，我听到他们在投射对自己不稳定的担忧：如果市场看不见的手判定我的技能不再有价值怎么办？如果我跌倒了谁会接住我？这一次，一个稀缺的雇员阶层——那些习惯了自动化别人、而不是被自动化的人——正在面对自己被淘汰的可能性。」

Jasmine Sun · 18 分钟 | **New York Times**

## 五、本周精选书籍

### Little Women

![Little Women](https://readwise-assets.s3.amazonaws.com/media/wisereads/articles/little-women/cover_image.jpg)

如果你对《小妇人》的印象只是一部关于四姐妹的甜美维多利亚小说，那你可能忘了 Marmee March 这个角色。Louisa May Alcott 以自己母亲为原型塑造了她，Marmee 贡献了书中最具现代感的几句台词：

「Jo，我几乎每一天都在生气；但我学会了不表现出来；我还希望能学会不去感觉到它，虽然这可能需要我再花四十年。」
