# 国外技术周刊 #142：Naval 预警软件正在归零

**欢迎打开本期国外技术周刊**：Every 的 Marcus Moretti 单人用 Claude Code 管理整个产品，不再写 ticket，对话即工作；David Epstein 新书揭示「单任务」才是成功的秘诀；Naval Ravikant 则预警 SaaS 只剩 18 个月，软件本身正在变成 commodity。

## 本周观点：软件正在归零，你的护城河在哪？

受 Naval Ravikant 一段播客内容启发——他抛出了一个让硅谷不安的论断：软件正在归零，SaaS 公司只剩 18 个月来证明自己不只是代码。

Naval 的逻辑很简单：AI 让软件的生产成本急剧下降，当每个人都能用自然语言生成任何应用时，代码本身就不再是竞争壁垒。Mustufa Khan 在 Twitter 上梳理了这层含义：「能活下来的公司不是那些软件最好的——软件正在归零。能活下来的是那些 AI 无法复制的东西：分发渠道、网络效应、数据飞轮、硬件整合、品牌、社区、监管深度。」

这不是危言耸听。Thoughtworks 最新技术雷达里对 AI agent 的警告同样印证了这点：prompt injection 让大多数 agent 默认不安全，Simon Willison 定义的「致死三要素」——私有数据、不可信内容、外部行动——意味着 agent 从设计上就是风险，而不是配置错误。安全本身正在变成一种新的护城河。

与此同时，产品管理的方式也在被 AI 重塑。Every 的 Marcus Moretti 单枪匹马运营写作产品 Spiral，他直接用 Claude Code 替代了整个产品团队：「对话就是工作」，不再写 ticket，不再纠结 SQL 里的分号。这不是降本增效，而是工作范式的根本切换。

当代码变得廉价，真正的稀缺品是什么？是品味、是分发、是信任、是那些在代码之外、需要时间才能积累的东西。如果你的公司只是一个 CRUD 应用加一层业务逻辑，AI 要复制你只是时间问题。但如果你拥有用户离不开的网络、一个认准你的品牌、一套对手进不来的数据飞轮——那才是真正的护城河。
